<h1 align= "center"> Todo List - Vanilla JS </h1>



 <p> <strong>Projeto:</strong> Aplicativo de lista de tarefas simples, criado usando <strong>HTML, CSS e JavaScript puro (Vanilla JS)</strong>.  </p>

---

<h2> Descrição </h1>

Este projeto é uma **To-do List** que permite ao usuário:

- Adicionar tarefas
- Marcar tarefas concluídas (visual)
- Excluir tarefas
- Salvar as tarefas no **LocalStorage** para que persistam mesmo após fechar o navegador  

O aplicativo não utiliza frameworks ou bibliotecas externas, sendo uma ótima prática para quem quer aprender **DOM Manipulation** e **JavaScript puro** ou quiser rever um conteúdo como eu fiz.

---

## Funcionalidades

1. **Adicionar tarefa:**  
   - Usuário digita a tarefa no input e clica no botão "Adicionar".  
   - A tarefa é armazenada no array de tarefas e exibida na lista.

2. **Excluir tarefa:**  
   - Cada tarefa possui um botão "Excluir".  
   - Ao clicar, a tarefa é removida do array e a lista é atualizada.

3. **Persistência de dados:**  
   - As tarefas são salvas no **LocalStorage**.  
   - Mesmo que o usuário feche ou atualize o navegador, as tarefas permanecem.

4. **Validação:**  
   - Não é permitido adicionar tarefas vazias.

---

## Tecnologias usadas

- **HTML5** – Estrutura da aplicação  
- **CSS3** – Estilização básica  
- **JavaScript (Vanilla JS)** – Lógica e manipulação do DOM  
- **LocalStorage** – Persistência de dados no navegador

## Como executar o projeto

1. Clone o repositório:

```bash
git clone https://github.com/seu-usuario/todo-list-vanilla-js.git
````

2. Abra o arquivo index.html no navegador. <br>
OBS: Não precisa instalar nada! Como o projeto é feito em Vanilla JS, tudo roda direto no navegador.
